## 💡 为什么会出现这个错误？

在 `camera_stream.py` 的 `stream_loop()` 方法中使用了 `CAM_FPS`：

```python
# 第 115 行附近
target_interval = 1.0 / CAM_FPS  # 使用配置文件中的 FPS
```

但是忘记在文件开头的 `import` 语句中导入它。

**MicroPython 的 import 规则**：
- ✅ 必须显式导入所有使用的变量
- ❌ 不会自动导入整个模块的所有内容
- ✅ 每个需要的变量都要在 import 列表中
