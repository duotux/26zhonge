## 📋 使用步骤（5 分钟完成）

### 步骤 1️⃣：准备音频文件

将你的 WAV 音频文件放到这个目录：
```
pc/assets/audio/
```

**如果没有音频文件？** 可以：
1. 使用录音软件录制（推荐 Audacity）
2. 或使用文本转语音工具（如 Balabolka）
3. 或从网上下载现成的提示音

**格式要求**：
- 文件扩展名：`.wav`
- 采样率：8000 Hz
- 位深：16-bit
- 声道：单声道

### 步骤 2️⃣：上传代码到 ESP32

在 Thonny 中打开并上传：
```
esp32/audio_player.py
```

### 步骤 3️⃣：运行 PC 端程序

```bash
cd pc
python main.py
```

**应该看到**：
```
[AudioServer] 启动于端口 5603
[AudioServer] 音频目录：pc/assets/audio
```

### 步骤 4️⃣：测试播放

在 ESP32 串口控制台输入：
```python
from audio_player import AudioPlayer
import uasyncio as asyncio

player = AudioPlayer()
asyncio.run(player.handle_command(b"warn1", level=1))
```

**如果听到声音** → ✅ 成功！

## 🔧 故障排查

### 问题 1：没有声音

**检查清单**：
- [ ] PC 端显示 AudioServer 已启动？
- [ ] `assets/audio/` 目录有 WAV 文件？
- [ ] ESP32 和 PC 能互相 ping 通？
- [ ] I2S 功放连接正确？

### 问题 2：连接超时

**解决方法**：
```python
# 在 ESP32 串口运行测试
import socket
sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
sock.connect(('192.168.137.1', 5603))
print("连接成功！")
```

### 问题 3：文件不存在

**检查**：
1. 文件名是否正确（区分大小写）
2. 路径是否匹配 `WARN_CMD_MAP` 中的配置
3. PC 端防火墙是否阻止

## 📝 配置文件说明

### ESP32 配置 (`esp32/config.py`)

```python
WARN_CMD_MAP = {
    b"warn1": "/audio/warn1.wav",   # ← 确保路径正确
    b"warn2": "/audio/warn2.wav",
    # ...
}
```

### PC 端配置 (`pc/core/config.py`)

```python
TCP_AUDIO_PORT = 5603  # ← 必须与 ESP32 一致
AUDIO_FILES_PATH = "pc/assets/audio"  # ← 音频目录
```
